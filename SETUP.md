# Flywire Employee Management App

A full-stack employee management system built with **Spring Boot (backend)** and **React + Material UI (frontend)**.

---

## 🚀 How to Run

### 1. Backend (Spring Boot)

**Prerequisites:** Java 17+, Maven  
**Steps:**

```bash
cd /"your-local-path"/flywire-exercise-main/
# Run the main class:
# com.flywire.exercise.FlywireSpringBootApp.java
```

- Starts the server at: `http://localhost:8080`

---

### 2. Frontend (React + MUI)

**Prerequisites:** Node.js, npm  
**Steps:**

```bash
cd /path/to/frontend
npm install
npm run dev
```

- Runs at: `http://localhost:5173`

---

## 🖥️ Features (Views)

1. **Active Employees** – List active employees  
2. **Employee Details** – View by ID  
3. **Hired Employees** – Filter by hire date  
4. **Create Employee** – Add new employee  
5. **Deactivate Employee** – Deactivate by ID

Each view connects to the backend via RESTful APIs.

---

## 📦 Tech Stack

- Backend: Spring Boot (Java 17)
- Frontend: React, TypeScript, Material UI
- API Communication: Axios

---

## ✅ Notes

- Ensure the backend runs before the frontend.
- The frontend expects the backend on `localhost:8080`.
